1. Where are your photos?

Pick the folder that contains the JPEGs you want to triage. Sub-folders are scanned recursively. Videos and RAW files are ignored automatically.

Advanced
If JPEGs live under JPEG/, enter that name.

2. Where should the project live?

A project folder keeps your decisions, caches, and config together, separate from the photos. You can move it later without affecting the photos.

Default: ~/PictureClassifier-Projects/<name>/.

3. How should scenes be grouped?

Scenes are how the photos are bucketed in the sidebar. Auto-suggestions (top 30% pick, bottom 30% reject) are computed within each scene. You can change this later without re-scoring.

minutes

4. Ready to create?

Review the settings. The first run scores every photo and clusters faces, which can take a few minutes for a thousand photos.
