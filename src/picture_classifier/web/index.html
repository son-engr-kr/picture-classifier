Picture Classifier

Score your photos for blur, exposure, and faces — then cull fast.

Open a project

Pick a folder containing your photos. The app scans for .jpg, .jpeg, .png and creates (or reloads) a picks.json file inside that folder.

Advanced
If your JPEGs live in a subfolder like JPEG/, enter its name.

Recent

Expected folder layouts

Both layouts are supported. You can switch how scenes are grouped (by folder vs. by capture-time gap) any time after opening.

Flat

📁 my-photos/
   🖼 IMG_0001.JPG
   🖼 IMG_0002.JPG
   🖼 IMG_0003.JPG
   …
   🗒 picks.json   (auto-created)

Use time-gap grouping later to auto-split into scenes.

Pre-grouped folders

📁 wedding-2025/
   📁 Scene_001/
      🖼 IMG_0001.JPG
      🖼 IMG_0002.JPG
   📁 Scene_002/
      🖼 …
   🗒 picks.json   (auto-created)

Folder names become scene names automatically.

Photo dir + JPEG subfolder

📁 shoot/
   📁 RAW/
   📁 JPEG/         ← set "JPEG subfolder"
      📁 Scene_001/
         🖼 IMG_0001.JPG
   🗒 picks.json   (at shoot/)

For workflows where RAWs and JPEGs live side-by-side.
